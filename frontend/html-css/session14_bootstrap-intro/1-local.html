Bootstrap Lokal

lokal olarak bağlantı için önce css linki head tagı içine, sonra js linki body kapanış tagı üzerine eklenir.

sadece bootstrap.css / bootstrap.min.css veya bootstrap.js / bootstrap.min.js lokalimizdeki dosya içine alsak ve link versek olur. diğer dosyalar başka tarayıcılar için düzenlenmiş css dosyalarıdır.
