--------- 1nci kurulum yöntemi

npm i -g typescript
kendi bilgisayarımıza global olrak kurulum yapar..

tsc -v
tsc --version
versiyonuna bakılır.

npm ls -g
npm içinde nereye kurulduğunu gösterir.

----------- 2nci kurulum yöntemi

npx create-react-app my-app --template typescript
yarn create react-app my-app --template typescript

react projesi olarak içinde typscript olan kurulum yapar
